# E-Commerce-Data-Analysis
This project analyzes an e-commerce dataset to identify sales trends, regional performance, and product demand. Using pivot tables, and charts, the data was cleaned, summarized, and visualized to generate insights. The analysis highlights top-performing regions, monthly sales trends, and order distribution to support data-driven business decisions.

Project Overview


This project focuses on analyzing an E-Commerce sales dataset to uncover valuable insights related to sales performance, regional trends, and product demand. The objective of this project is to transform raw sales data into meaningful insights using data analysis and visualization techniques.
By using Microsoft Excel, Pivot Tables, and charts, the dataset was cleaned, analyzed, and visualized to understand business performance and support data-driven decision making.

🎯 Business Questions

This analysis answers several important business questions:
Which region generates the highest revenue?
What are the monthly sales trends?
Which product categories perform best?
What is the distribution of order statuses?
How do different regions contribute to total sales?

🗂 Dataset Description
The dataset includes the following key fields:
Order ID – Unique identifier for each order
Order Date – Date when the order was placed
Region – Sales region of the order
Product Category – Category of product purchased
Sales / Revenue – Total revenue generated
Order Status – Status of the order (Completed, Cancelled, etc.)

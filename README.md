# 📊 E-Commerce Sales Analysis (Excel Project)

## Project Overview
This project analyzes an e-commerce sales dataset to identify revenue trends, product performance, and order fulfillment patterns using Excel.

The objective of this project is to demonstrate core data analytics skills including data preparation, pivot table analysis, and data visualization.

---

## Dataset
The dataset contains order-level information including:

- Order ID
- Product Name
- Category
- Region
- Sales Representative
- Quantity
- Unit Price
- Order Status

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Data Visualization (Charts)

---

## Analysis Performed
The following analysis was conducted:

- Calculated total revenue using Quantity × Unit Price
- Classified orders into High Value and Standard Value
- Analyzed revenue by region
- Evaluated product performance
- Analyzed sales representative performance
- Visualized order status distribution

---

## Key Insights
- Electronics category generated the highest revenue
- Majority of orders were successfully shipped
- High-value orders contribute significantly to overall revenue
- Some regions outperform others in total sales

---

## Dashboard
A simple Excel dashboard was created to summarize the analysis including:

- Revenue KPIs
- Category revenue comparison
- Order status distribution
